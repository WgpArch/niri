# 🏠 Welcome to my Niri Config

A scrollable-tiling Wayland compositor setup on Arch Linux, configured using the KDL markup language. This setup features a unique **Dual Waybar** layout to perfectly accommodate 11 workspaces with custom icons, alongside comprehensive system metrics.

![Niri Showcase](screenshots/screenshot_2026-05-24_19-50-43.png)

## ✨ Highlights
- **Dual Waybar Setup:** A centered top bar for system info and a floating bottom dock for 11 icon-based workspaces.
- **Catppuccin Macchiato:** Themed with Mauve active borders and matching Waybar CSS.
- **Extensive Window Rules:** Pre-configured floating, sizing, and opacity rules for apps like Floorp, OBS, and Kitty.
- **XWayland Satellite:** Seamlessly handles X11 applications via `xwayland-satellite`.
