# ⌨️ Keybinds

Here are the default keybindings for this Niri setup. The main modifier key is `Mod` (Super/Windows key).

## 🖥️ Programs & System

| Keybind | Action |
| :--- | :--- |
| `Mod + T` | Open Terminal (`lxterminal`) |
| `Mod + A` | Open App Launcher (Rofi) |
| `Mod + X` | Open Power Menu (Rofi) |
| `Mod + E` | Open File Manager (`pcmanfm`) |
| `Mod + Q` | Close focused window |
| `Mod + V` | Quit Niri |
| `Super + Alt + L` | Lock Screen (`hyprlock`) |

## 📸 Screenshots

| Keybind | Action |
| :--- | :--- |
| `Mod + Shift + S` | Screenshot (Custom script) |
| `Mod + Ctrl + S` | Screenshot interactive selection (`grim`/`slurp`) |
| `Mod + P` | Screenshot entire screen |
| `Alt + P` | Screenshot selected area |
| `Ctrl + P` | Screenshot active window |

## 🔍 Focus & Movement

| Keybind | Action |
| :--- | :--- |
| `Mod + Arrow Keys` | Focus column/window |
| `Mod + Home / End` | Focus first/last column |
| `Mod + Ctrl + Left/Right` | Focus workspace up/down |
| `Mod + Shift + Arrow Keys` | Move column/window |

## 🗂️ Workspaces

| Keybind | Action |
| :--- | :--- |
| `Mod + [1-0]` | Focus workspace 1-10 |
| `Mod + Alt + [1-0]` | Move window to workspace 1-10 |
| `Mod + Wheel Scroll` | Cycle workspaces up/down |

## 📐 Layout & Sizing

| Keybind | Action |
| :--- | :--- |
| `Mod + R` | Switch preset column width |
| `Mod + Shift + R` | Switch preset window height |
| `Mod + M` | Maximize column |
| `Mod + C` | Center column |
| `Mod + Minus / Equal` | Set column width -5% / +5% |
| `Mod + Shift + Minus / Equal` | Set window height -5% / +5% |
| `Mod + BracketLeft / Right` | Consume/Expel window left/right |
| `Mod + Comma / Period` | Consume/Expel window into/from column |
| `Mod + W` | Toggle window floating |
| `Mod + K` | Switch layout |

## 🔊 Media Keys

| Keybind | Action |
| :--- | :--- |
| `Volume Up/Down` | Volume 5% +/- (`wpctl`) |
| `Shift + Volume Up/Down` | Volume 1% +/- (Fine control) |
| `Mute` | Toggle audio mute |
| `Brightness Up/Down` | Backlight 5% +/- (`brightnessctl`) |
